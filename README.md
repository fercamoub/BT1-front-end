# Breakable toy Front-End

## Features

- Modern UI built with React
- Responsive design for all devices
- Integration with the DestroyableToy back-end API

## Getting Started

### Prerequisites

- Node.js
- npm or pnpm

### Installation

```bash
git clone https://github.com/your-username/DestroyableToy-front-end.git
cd BT1-front-end
npm install
```

### Running the App

```bash
npm start
```
